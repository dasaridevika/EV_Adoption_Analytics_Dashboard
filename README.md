# 🚗 EV Adoption Analytics Dashboard

## 📌 Project Overview

The EV Adoption Analytics Dashboard is a Data Analytics project designed to analyze the factors influencing electric vehicle (EV) adoption among consumers. Using a dataset of 50,000 records, the project explores how income, environmental awareness, range anxiety, home charging availability, education level, and other demographic factors affect EV adoption likelihood.

The dashboard provides interactive visualizations and business insights to help understand consumer behavior and support data-driven decision-making.

---

## 🎯 Objectives

* Analyze consumer EV adoption patterns.
* Identify key drivers of EV adoption.
* Compare adoption trends across demographic groups.
* Visualize insights using interactive dashboards.
* Support business and policy decisions through data analysis.

---

## 📊 Key Insights

* Higher annual income is associated with higher EV adoption.
* Environmental awareness positively impacts EV adoption.
* Range anxiety significantly reduces EV adoption likelihood.
* Home charging availability encourages EV adoption.
* Technology affinity and EV knowledge contribute to adoption decisions.
